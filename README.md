A production-grade conversation intelligence platform that combines Retrieval-Augmented Generation (RAG), topic segmentation, persona extraction, and hybrid semantic search to transform raw conversation datasets into an interactive knowledge discovery system.

The platform enables users to explore conversations, retrieve contextual information, identify topic transitions, extract user personas, and interact through a conversational AI interface.

---

# 📌 Overview

RAG-Based Conversation Intelligence System is an enterprise AI platform designed to analyze and understand large-scale conversation datasets.

The system processes messages chronologically, detects topic boundaries dynamically, generates topic summaries, extracts evidence-backed persona traits, and enables semantic retrieval using hybrid search techniques.

It combines dense vector search, keyword retrieval, and LLM-powered reasoning to provide accurate and context-aware responses.

# 🚀 Getting Started

## 📦 Prerequisites

- Python 3.10+
- OpenAI API Key
- FAISS
- Docker (Optional)

---

## 📥 Clone Repository

```bash
git clone https://github.com/devanshnegi88/rag-conversation-intelligence.git

cd rag-conversation-intelligence
```

## 🐍 Create Virtual Environment

```bash
python -m venv venv
```

### Windows

```bash
venv\Scripts\activate
```

### Linux/macOS

```bash
source venv/bin/activate
```

## 📦 Install Dependencies

```bash
pip install -r requirements.txt
```

## ⚙️ Configure Environment Variables

```env
OPENAI_API_KEY=your_api_key
```

## ▶️ Run Application

```bash
python main.py
```

# 🏗️ System Workflow

1. Load conversation dataset.

3. Detect topic transitions.
4. Generate topic summaries.
5. Extract persona traits.
6. Generate embeddings.
7. Build FAISS index.
8. Enable hybrid retrieval.
9. Launch chatbot interface.
10. Serve contextual responses.
